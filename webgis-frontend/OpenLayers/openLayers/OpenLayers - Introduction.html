介绍

目标

OpenLayers是一个模块化、高性能、功能丰富的库，用于在网页上创建交互式地图。

该库支持大量的矢量数据格式。有了OpenLayers的地图投影支持，数据可以用于任何投影。

公共API

OpenLayers作为ol npm package，提供官方支持的所有的API所有模块。.

渲染器和浏览器支持

默认情况下，OpenLayers使用性能优越的画布渲染器进行渲染。

OpenLayers可以运行在支持HTML5和ECMAScript 5的所有现代浏览器上，包括Chrome, Firefox, Safari 和 Edge。

该库可以用于桌面、笔记本和移动设备，它支持指针交互和触摸交互。

模块及命名规范

OpenLayers默认使用“驼峰式命名法”为模块命名导出，并且可以包含附加常量或函数作为命名导出:

import Map from 'ol/Map';
import View from 'ol/View';

在包的子文件夹中提供了按父类分组的类层次结构，例如layer/。

为了方便起见，这些也可用作命名导出，例如：

import {Map, View} from 'ol';
import {Tile, Vector} from 'ol/layer';

除此之外，具有小写名称的模块还提供常量或函数作为命名导出:

import {inherits} from 'ol';
import {fromLonLat} from 'ol/proj';
